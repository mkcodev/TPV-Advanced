# next-TPV

TPV/POS moderno para hostelería (bares y restaurantes) en España.
Uso real (el bar) + producto vendible (SaaS multi-negocio).

## Documentación (leer antes de tocar código)
- `CLAUDE.md` — constitución del proyecto (stack, reglas, convenciones).
- `docs/PLANIFICACION-TPV.md` — visión, funcionalidades, roadmap, legal.
- `docs/DATABASE-SCHEMA.md` — modelo de datos (fuente de verdad).
- `docs/DESIGN-SYSTEM.md` — tokens, componentes y reglas estrictas de UI.
- `docs/PASOS-A-SEGUIR.md` — guía lineal con todos los pasos en orden (empieza aquí).
- `docs/CLAUDE-CODE-SETUP.md` — cómo continuar el desarrollo en Claude Code.
- `docs/ROADMAP-Y-PROMPTS.md` — fases, tareas y prompts listos para Claude Code.

pnpm install
